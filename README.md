# 🏥 Hospital Management System

A basic Hospital Management System developed to practice and apply **Object-Oriented Programming (OOP)** concepts. The project simulates simple hospital operations by organizing data and functionality into classes and objects.

## Project Objective

The primary goal of this project was to strengthen my understanding of OOP principles, including:

* Classes and objects
* Encapsulation
* Abstraction
* Inheritance *(if implemented)*
* Polymorphism *(if implemented)*
* Modular program design

## Features

* Manage basic patient information
* Store and display hospital records
* Menu-driven user interaction
* Simple and organized class structure

## Technologies Used

* Java
* Visual Studio Code

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/yourusername/hospital-management-system.git
```

2. Open the project in Visual Studio Code.

3. Run the main program file.
